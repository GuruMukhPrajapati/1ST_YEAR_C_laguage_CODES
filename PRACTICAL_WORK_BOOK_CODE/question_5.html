Using Special Characters in HTML

The non-breaking space is used to add extra space between words or letters, and is represented by the entity code " ".

The ampersand is used to represent HTML entities, and is represented by the entity code "&".

The less-than symbol is used to denote the beginning of an HTML element, and is represented by the entity code "<".

The greater-than symbol is used to denote the end of an HTML element, and is represented by the entity code ">".

Double quotes are used to enclose attribute values in HTML elements, and can be represented using either the entity code """ or single quotes.
